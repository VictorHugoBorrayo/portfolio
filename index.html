Portafolio

Predicción de cobertura en red mesh

Se desarrolló un ETL en premisas que automatizó la extracción y almacenamiento en un DataLake. Con técnicas estadísticas y de Machine Learning se identificaron anomalías e imputaron. Los datos limpios del Data Mart se enriquecieron con información de altimetría e imagenes satelitales proveídas por API's de Google. Se aplicó ingeniería de características para mejorar la precisión de los modelos de Machine Learning y de Deep Learning. Se utilizó MLFlow para el seguimiento de los experimentos y el análisis final de los resultados.

Ingeniería de datos con Azure

Participación en el desarrollo y mantenimiento de un sistema de ingeniería de datos que utiliza Function Apps y Synapse para orquestrar el ETL de que nutre un DataWarehouse. Existen ciertas automatizaciones que utilizan eventos para la ejecución de determinados pipelines cuando se han alojado los datos de origen en su sitio. Durante el desarrollo se utilizan pipelines de CICD que permiten la integración eficiente de los cambios en los distintos ambientes.

Aplicativo para recolección datos en campo

Aplicativo montado en la nube de Azure haciendo uso de Container Registry, WebApp, Base de datos serverless y blob storage. Se automatizó la labor de recolección de datos de cuadrillas de técnicos realizando trabajos en campo, ordenación de fotografías y seguimiento de materiales. Con lo anterior se crearon herramientas para la gestión de la aplicación y el análisis de datos con herramientas como Power Bi.

Aplicativo para empresa de contabilidad

Utilizando Flask y App Service de Azure se desarrollo un aplicativo customizable por medio de tablas de configuración y registros No SQL. Se eficientó la toma de datos y se automatizó la generación de reportes, los cuales son dinámicos y se pueden customizar con plantillas Json. Para asegurar la seguridad se utilizó Azure VPN Gateway para el acceso a la aplicación de parte de los usuarios.
